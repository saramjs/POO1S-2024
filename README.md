# POO1S-2024
Programação Orientada a Objeto 
A classe Calculadora é responsável por fornecer funcionalidades básicas de uma calculadora. Ela suporta operações de adição, subtração, multiplicação e divisão entre dois números inteiros. Além disso, possui um método para verificar se um número é primo. Esta classe é utilizada em conjunto com a classe Main para demonstrar o uso dessas operações.
